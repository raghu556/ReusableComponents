Autocomplete with pagination Component

Component Description:

  • This component allows autocomplete functionality with pagination based on provided array
  • Based on current search text, search results and pagination will get updated.
  • Autocomplete results will be retrieved even with single letter search.
  • When you select any item in search result box, the selected item gets populated in search text box.

Component Key Features:

  • This component basically provides combined features of two functionalities, one is autocomplete and another one is pagination
  • User can cusomize the array or assigned dynamic array
  • User can customize the no of items to be displayed as per choice using "pageSize" variable
  • User can select only one item and selected item is displayed below the search text box.

Variables need to cutomise are

  • movies array
  • pageSize
